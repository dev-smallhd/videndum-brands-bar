Adding Videndum Brands bar to a website:
1. Include the code starting at "<div class="videndum-section">..." at the start of your webpage's <body> tag.
2. Add latest version of jQuery library (Skip if already included on your website).
2. Add scripts from js/scripts.js to your scripts file.
3. Add styles from css/styles.css to your stylessheet file.
4. Copy/Update all images from /images folder.
5. Skip adding inline css reset if css reset is already used.
